PROCESS DESCRIPTIONS:
====================

1. Access Application (Sign up/Login)
   - Handles user authentication
   - Manages login/signup flows
   - Validates user credentials
   - Redirects to main app

2. Discover Recipes (Search/Filter/Explore)
   - Provides recipe search functionality
   - Implements filtering by category, cuisine, difficulty
   - Fetches data from external API
   - Displays search results

3. Manage Favorites (Add/Remove/View Favorites)
   - Allows users to add recipes to favorites
   - Provides remove functionality
   - Displays user's favorite recipes
   - Syncs with cloud storage

4. Manage Saved Recipes (Add/Remove/View Saved)
   - Enables saving recipes for later
   - Provides remove functionality
   - Displays saved recipe collection
   - Syncs with cloud storage

5. View Recipe Details (Ingredients/Instructions/Nutrition)
   - Shows detailed recipe information
   - Displays ingredients and measurements
   - Shows step-by-step instructions
   - Provides nutritional information

6. Profile Management (View/Edit Profile)
   - Displays user profile information
   - Allows profile editing
   - Manages user preferences
   - Handles account settings

7. Recipe Categories (Browse by Category/Area)
   - Provides category-based browsing
   - Shows recipes by cuisine area
   - Implements category filtering
   - Displays categorized results

DATA STORE DESCRIPTIONS:
========================

D User Data (Profiles, Authentication)
- Stores user account information
- Manages authentication credentials
- Contains user profile details
- Handles user preferences

D Recipe Data (Meals, Categories, Ingredients)
- Stores recipe information
- Contains meal categories
- Manages ingredient data
- Handles recipe metadata

D User Preferences (Favorites, Saved Recipes)
- Stores user's favorite recipes
- Manages saved recipe collection
- Handles user-specific data
- Syncs across devices

D External API Data (TheMealDB)
- External recipe database
- Provides recipe information
- Contains meal categories
- Supplies ingredient data
